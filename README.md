# markov-localozation-2Dmap
2D карта гексагонов, по которой перемещается робот. Имеется сенсор, который умеет определять цвет гексагона под собой, а также наличие стены перед собой. Локализация приближена к реальности возможностью ошибчной детекции и погрешностью поворота и передвижения.

## установкка
Для запуска робота необходимо:
1) клонировать репозиторий
2) установить в окружении библиотеки numpy, pygame

### Робот управляется при помощи клавиш:
1) стрелки влево, вправо - повороты
2) стрелка вверх - движение
3) стрелка вниз - использовать сенсор
4) клавиша t - переместить робота в рандомную точку, сбросить карту вероятностей
5) esc - завершить симуляцию
